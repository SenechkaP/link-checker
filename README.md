Дата выполнения: 05.12.2025
Версия go: 1.25.3

Для возвращения pdf документа использовал пакет github.com/jung-kurt/gofpdf

Выбрал этот пакет, потому что быстро с ним разобрался и нашел пример использования: https://products.fileformat.com/ru/pdf/go/gofpdf/

Использовал технику graceful shutdown; при получении сигналов SIGTERM или SIGINT сервис прекращает принимать запросы, при этом дожидаясь завершения принятых до остановки веб-сервера запросов.

WaitGroup, созданная в main.go, никак не относится к graceful shutdown; я создаю эту WaitGroup в main (а не внутри service) для того, чтобы при остановке сервиса точно дождаться завершения всех созданных горутин в слое service.

## Запуск сервиса
В терминале в корневой директории проекта:
```
go run ./cmd
```

## Примеры запросов
+ Получение статусов интернет-ресурсов
```
http://localhost:8080/links/statuses
```
```
{
  "links": [
    "https://ya.ru",
    "https://hh.ru"
  ]
}
```
+ Получение pdf файла с запрошенными nums
```
http://localhost:8080/links/pdf
```
```
{
  "links_list": [1, 2]
}
```
